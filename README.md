 Sleep and Health Matrix Analysis

This project analyzes sleep patterns and health metrics such as heart rate, stress level, and BMI using machine learning to predict whether a person is healthy or unhealthy.
Technologies
 1.Python
 2.Pandas
 3.NumPy
 4.Scikit-learn
 5.Streamlit
 6.Matplotlib
 
  Features
1. Sleep pattern analysis
2.Health status prediction
3. Machine learning model (Random Forest)
4. Data visualization
5.Interactive web interface

 How to Run the Project
1. Install Python
2. Install required libraries
3. Run the Streamlit application

```bash'''
pip install -r requirements.txt
streamlit run main.py
